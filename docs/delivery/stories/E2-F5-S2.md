# Remove em dashes from the report output

## Goal

Remove em dashes from the report output, replacing them with clearer punctuation.

## Definition of Done

- No em dash (either the literal character or `&mdash;`) appears in the generated
  `report.html`.
- Prose em dashes become commas/colons/parentheses; em dashes used as empty-value
  placeholders in tables become a plain "-" or "n/a".

## Acceptance Criteria

```gherkin
Scenario: The generated report has no em dashes
  Given a freshly generated report.html
  When it is scanned for the em-dash character and &mdash;
  Then none are found

Scenario: Empty-cell placeholders remain readable
  Given a table cell that previously showed an em dash for "no value"
  When the report renders
  Then it shows a plain "-" (or "n/a") instead
```

## Test Plan

Regenerate `report.html`; grep the output for the em-dash character and
`&mdash;` and assert zero matches. Scope is `HTML_TMPL` copy only, not the
Python source comments.
