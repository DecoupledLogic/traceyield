# Cross-provider aggregation

## Goal

Un-scope `aggregate()` from `provider='claude'` so all providers flow through,
with a `by_provider` breakdown, while preserving the Claude equivalence oracle.

## Definition of Done

- `aggregate(conn, provider=None)` spans all providers; `by_provider` appears in
  the day/session payload.
- `pricing_history.json` carries a provider dimension with backward-compatible reads.
- `aggregate(conn, provider='claude')` reproduces `analyze()` turn-for-turn.

## Acceptance Criteria

- [ ] With both providers in `usage.db`, day/session totals include Codex
- [ ] `aggregate(conn, provider='claude')` reproduces `analyze()` exactly
- [ ] Day/session top-level shape is unchanged (provider is a sub-facet)

## Test Plan

A mixed-provider fixture asserting combined and `by_provider` totals; the
equivalence oracle pinned to `provider='claude'`.
