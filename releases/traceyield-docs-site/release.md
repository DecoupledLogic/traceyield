TraceYield now lives on the web. Instead of cloning the project to read what it
does, you can open the site, browse the documentation, and search across all of
it from anywhere.

## What's new

The documentation is now a single, organized site rather than a scattering of
files. The pages are grouped so you can find the overview, the data model, and
the provider guides without hunting, and a search box across the top finds any
topic in a keystroke.

## What to do

Nothing — just visit the site. If you had the old links bookmarked, the new home
is `decoupledlogic.github.io/traceyield`, and search will get you the rest of the
way.
