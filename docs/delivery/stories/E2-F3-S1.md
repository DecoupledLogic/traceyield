# Cost-by-provider panel and provider filter/toggle

## Goal

Add a "Cost by provider" bar and a provider filter/toggle to the client-side
`aggregate()` in the emitted HTML.

## Definition of Done

- `by_provider` renders as a bar (like "Cost by model").
- A provider filter re-aggregates the whole view; All/Claude/Codex are selectable.

## Acceptance Criteria

```gherkin
Scenario: Cost by provider renders
  Given a payload with by_provider data
  When the report loads
  Then a "Cost by provider" bar shows each provider's cost

Scenario: The provider filter scopes the view
  Given the report with both providers
  When the user selects "Codex"
  Then all panels re-aggregate to Codex-only figures
```

## Test Plan

Manual smoke of `report.html`; verify totals reconcile between the All and
per-provider selections.
