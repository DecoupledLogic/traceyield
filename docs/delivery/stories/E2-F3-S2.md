# Token-normalized neutral-currency comparison view

## Goal

Add a token-normalized cross-provider comparison so dollars aren't the only way
to compare providers (Decision 0007 D4).

## Definition of Done

- A view compares providers by tokens (the neutral currency).
- Combined dollar totals are labelled as a per-provider projection.

## Acceptance Criteria

```gherkin
Scenario: Token comparison across providers
  Given both providers present
  When the neutral view is shown
  Then providers are compared by normalized token counts independent of pricing

Scenario: Combined dollars are labelled a projection
  Given a combined dollar total spanning providers
  When it is displayed
  Then the UI marks it as dependent on per-provider rate cards
```

## Test Plan

Manual smoke; confirm the token view matches the summed token vectors and the
dollar-projection label is present.
